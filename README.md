# Sarbjot Singh — IT Tools Portfolio

Full-stack internal tooling built for large-scale enterprise IT operations.
All apps are SharePoint-hosted, vanilla JS + HTML/CSS with no build step.

## Projects

| Project | Description | Tech |
|---|---|---|
| [IT Hire Dashboard](./it-hire-dashboard/) | New hire onboarding tracker with bulk CSV import | HTML, JS, SharePoint REST API |
| [New Hire Dashboard](./new-hire-dashboard/) | Multi-region hire onboarding & credential management | HTML, JS, Chart.js |
| [Quiz Platform](./quiz-platform/) | Admin + Agent quiz system with live scoring | HTML, JS, SharePoint Lists |
| [Headset Tracker](./headset-tracker/) | Equipment inventory & assignment management | HTML, JS |
| [Onboard / Offboard System](./onboard-offboard-system/) | Employee lifecycle management portal | HTML, JS |
| [Employee Recognition App](./employee-recognition/) | Peer-to-peer recognition ("High Five") system | JS, SharePoint REST API |
| [Windows 11 Deployment](./windows-deployment/) | Zero-touch OS install + Entra auto-enrollment via WCD provisioning package | XML, WCD, Entra ID |

## Skills Demonstrated

- **Frontend**: Fluent UI design system, dark mode, responsive layouts, keyboard-accessible forms
- **Data**: CSV bulk import/export, inline editing, real-time filtering, analytics dashboards
- **SharePoint**: REST API (GET/POST/PATCH/DELETE), PnP JS, list operations, permission handling
- **UX**: Search with suggestions, confirmation modals, status badges, loading states

## Notes

- All SharePoint tenant URLs have been replaced with `your-tenant.sharepoint.com` placeholders
- Company-identifying information has been removed for privacy
- Apps are designed to run inside a SharePoint site with appropriate list structures
